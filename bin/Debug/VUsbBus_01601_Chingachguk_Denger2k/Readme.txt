    +-------------------------------------------------------------------------+
    |ИНСТРУКЦИИ ПО УСТАНОВКЕ [Virtual USB bus driver v.0.16.0.1; HL&SRM mod +]|
    +-------------------------------------------------------------------------+

	1. Если ранее стояли какие-либо эмуляторы HASP, необходимо их 
	  удалить, пользуясь сопроводительной документацией к этим эмуляторам.
	2. Для очистки определенных веток реестра запустить файл 
	  UnInstall_old_reg.bat, при необходимости - очистить реестр вручную.
        3. Устанавливаем программу(ы);
	4. Запускаем "install.bat" (установка драйвера эмулятора);
	5. Из папки \reg.files вносим данные соответствующего(их)
	  reg-файла(ов) в реестр.
	6. Перезагружаем компьютер;
        7. Работаем.

З.Ы.:	А) Данный эмулятор протестирован с HASP Device Driver v.4.104.5.24
	Б) Папка \reg.files содержит модифицированные для данного VUsbBus.sys
	  дампы на следующие программы:

		1. Kompas v.9(all SP),v.10 SP1/2
		2. APM WinMachine v.8.x, v.9.4
		3. T-Flex v.11.0.18/0.23/(0.25/0.26 - separate thanks elite_r)
		4. Potok 2005
		5. Астра-Нова 2005, 2007

	  - т.е. при использовании соответствующих reg-файлов с настоящим
	  эмулятором работоспособность этих программ будет сохранена.
	В) Дополнительная информация:
		remove.bat		- удаление драйвера эмулятора;
		!restartemulator.bat 	- перезапуск драйвера эмулятора.
	Г) За эмулятор спасибо Chingachguk & Denger2k.
