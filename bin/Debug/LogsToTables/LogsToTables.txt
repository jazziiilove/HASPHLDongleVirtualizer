Конвертер логов предназначен для подготовки таблиц к табличному эмулятору из логов
логгеров, из таблиц выдранных из "конверта" и записи готовых таблиц либо в файл либо в реестр.
Возможность добавлять или заменять таблицы прямо в реестре доступна лишь для
той ветки реестра где установлены соответствующие ключи для Hasp HL (Dongletype, type)
ну и ест-но только для мультикея, остальные ветки тоже найдёт - но можно только удалять.
При выборе закладки "Реестр" поиск в реестре автоматический, но можно и в ручную.
Но помните сменились пароли ключа, уже таблицы не вставить(стоит проверка на валидность паролей)
Ну и наоборот после выбора ветки в реестре возможна обработка логов только для этого пароля.
Естесвенно есть очистка. и чаще пользуйтесь контекстным всплывающим меню :-)
Удачи в нелёгком пути народного образования (Ибо сей продукт только для оного и предназначен)
